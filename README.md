# mim-zap

Man in the Middle HTTP proxy for automated OWASP Zed Attack Proxy (ZAP) inspection in a container.

## Dependencies

| Name                                          | Purpose                            |
|-----------------------------------------------|------------------------------------|
| [zaproxy](https://github.com/zaproxy/zaproxy) | zaproxy - the actual security tool |
